Exporting to PNG

  1. Open the ad's HTML file in Chrome.
  2. Open DevTools (Cmd+Opt+I) → Elements panel.
  3. Select the .frame div (the inner 1080×H wrapper).
  4. Cmd+Shift+P → type "Capture node screenshot" → Enter.
  5. PNG downloads at exact pixel dimensions.
